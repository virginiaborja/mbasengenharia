# MBAS Engenharia 🏗️

Professional website for **MBAS Engenharia**, the civil-engineering practice of **Márcia Barbosa** — a civil engineer (UFMG) with 30+ years of experience managing and executing industrial, commercial, and residential construction projects in Brazil. The site was designed and developed by [Virginia Borja](https://virginiaborja.github.io/webdeveloper/) as a client project.

> 🇧🇷 The site content is written in Brazilian Portuguese, matching the company's local audience.

## 🌐 Live Site

**[virginiaborja.github.io/mbasengenharia](https://virginiaborja.github.io/mbasengenharia/)**

## ✨ What's on the Site

A clean, multi-page business site in a navy-and-sand palette:

- **Home (`index.html`)** — services offered (construction management and execution, budgeting, technical inspections, ESG certifications), a portfolio overview with cards linking to each category of work, an "About Me" section on Márcia's background, and contact links (WhatsApp, e-mail, LinkedIn)
- **Obras Industriais (`industriais.html`)** — featured industrial project: **Arena MRV**, the 46,000-seat home stadium of Clube Atlético Mineiro in Belo Horizonte, with a photo gallery
- **Obras Comerciais (`comerciais.html`)** — featured commercial project: the **Olegário Jardins** restaurant on Av. Prudente de Moraes, Belo Horizonte, with a photo gallery
- **Obras Residenciais (`residenciais.html`)** — featured residential project: the **Casa São Bento** high-end 3-story home, with a photo gallery

## 🛠️ Built With

- **HTML5** — semantic multi-page structure
- **CSS3** — a single shared stylesheet (`css/styles.css`) with flexbox/grid layouts, portfolio cards, and responsive image grids
- **Simple Icons** (via jsDelivr CDN) — WhatsApp, Gmail, and LinkedIn contact icons

No frameworks, no build step — a fully static site.

## 🚀 Running Locally

No installation needed:

```bash
git clone https://github.com/virginiaborja/mbasengenharia.git
cd mbasengenharia
```

Then open `index.html` in your browser and navigate from there.
